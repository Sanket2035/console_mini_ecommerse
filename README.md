# Mini E-Commerce Console Application 🛒

A lightweight, terminal-based e-commerce platform engineered to simulate a complete digital storefront. Built entirely with Core Java and JDBC, this application demonstrates robust backend logic, strict Object-Oriented Programming (OOP) principles, and secure relational database management.

## 🚀 Features

* **Interactive Terminal Interface:** Streamlined command-line menus for seamless product browsing, shopping cart management, and checkout flows.
* **Database Integration:** Secure relational database layer utilizing JDBC for real-time product inventory updates and accurate data retrieval.
* **CRUD Operations:** Full Create, Read, Update, and Delete functionality for managing user profiles and product catalogs.
* **Robust Error Handling:** Comprehensive exception handling and input validation to prevent system crashes and invalid data entry during user transactions.

## 🛠️ Technologies Used

* **Language:** Core Java (JDK 11+)
* **Database API:** JDBC (Java Database Connectivity)
* **Database:** PostgreSQL / MySQL
* **Architecture:** Monolithic Console Application using OOP principles

## 📋 Prerequisites

Before you begin, ensure you have the following installed on your local machine:
* [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) (Version 11 or higher)
* [PostgreSQL](https://www.postgresql.org/download/) or [MySQL](https://dev.mysql.com/downloads/)
* The appropriate JDBC Driver (`postgresql-x.x.x.jar` or `mysql-connector-java-x.x.x.jar`)
